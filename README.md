# PRODIGY_DS_01



## Task 1: Data Visualization



This repository contains Task 01 of the Prodigy InfoTech Data Science Internship.



### Problem Statement

Create a bar chart or histogram to visualize the distribution of a categorical or continuous variable, such as population data.



### Dataset

The dataset used in this task contains country-wise population data collected from the World Bank.



### Work Done

- Loaded and cleaned the population dataset
- Selected relevant data for analysis
- Created a bar chart to compare population across countries
- Created a histogram to visualize population distribution
- Observed patterns and trends in the data


### Tools & Technologies

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
